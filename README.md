# ✏️ Handwritten Equation Solver

A deep learning project that recognizes handwritten math equations and solves them in real time!

## 🎯 What it does
- User draws a math equation on a canvas
- CNN model recognizes each symbol
- App solves the equation and displays the answer!

## 🧠 Model
- Architecture → CNN (TensorFlow + Keras)
- Training accuracy → 89.47%
- Validation accuracy → 95.89%
- Classes → 0-9, +, -, ×, ÷

## 🛠️ Tech Stack
- Python
- TensorFlow + Keras
- OpenCV
- Streamlit
- Streamlit Drawable Canvas

## 📊 Dataset
- Handwritten Math Symbols Dataset (Kaggle)
- 8384 images across 15 classes

## 🚀 How to run locally
pip install -r requirements.txt
streamlit run app.py
